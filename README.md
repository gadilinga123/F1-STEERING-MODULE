## F1-Style Steering Wheel Module

**Context:**  
Developed as part of a master’s coursework based on a problem statement provided by the Ferrari engineering team.

### Key Features
- Designed support for both **analog and digital input interfaces**
- Integrated a **TFT display with SPI protocol** for real-time data visualization
- Implemented **CAN communication** with a PC (send & receive)
- Enabled **dual power operation**:
  - 12 V automotive supply
  - 5 V USB input
- Applied **automotive-grade protection concepts** for power inputs
